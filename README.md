
# MLWBD Bypass

This script can bypass MLWBD ad links and give you direct download link for selected movie. 


## Installation

Need python 3

```bash
pip install requests bs4 colorama

```





## Features

- Colorful
- Fast
- No rate limit
- Proxyless

## How to use: 
- Visit MLWBD and copy link like https://mlwbd.fyi/movie/nowhere-2023/
```bash
Run python main.py

```
- Wait few seconds :D
